onDidLoadPairAsset:

Called when the the PairAsset is added to the view and completes loading. 

@optional

- (void)onDidLoadPairAsset:(nonnull PairAsset *)pairAsset;
Parameters
pairAsset

The newly loaded PairAsset.

Discussion

If the PairAsset was loaded with a PairAssetLoader, this delegate function will be called after the model file has been completely loaded. If the PairAsset was loaded directly from on an on disk URL it will be called immeditely.
