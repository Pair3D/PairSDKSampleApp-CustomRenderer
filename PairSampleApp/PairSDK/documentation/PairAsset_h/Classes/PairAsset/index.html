Superclass:

Declared In:
PairAsset.h

Introduction

PairAsset are added to the PairView. They include a representation of a 3D model.

Discussion

Use Pair's custom model format to load 3D mesh and texture information, as well as optional swappable swatches.

initWithPairAssetLoader:

Initializes a PairAsset using the specified PairAssetLoader. 

- (nullable instancetype)initWithPairAssetLoader:(nonnull id<PairAssetLoaderProtocol>)pairAssetLoader;
Parameters
pairAssetLoader

A loader object, conforming to the PairAssetLoaderProtocol, that will be used to asyncronously load the PairAsset model.

Return Value

A new PairAsset object.

Discussion

While the asset is loading a placeholder wireframe bounding box will be displayed in it's place.
